[CMD] ip command

Posted on 2019-10-20

ip命令在linux下可以通过安装net-tools获得。ip（Internet Protocol）命令可以用来展示或者修改系统的路由，设备和隧道灯信息。linux下还有一个相似的命令ifconfig（deprecated），与之相比ip命令可以完成更多的功能。

ip命令可以用于配置修改路由设置，设置隧道，查询ip地址和其他属性信息，修改接口的状态以及添加，修改或删除ip地址和路由等。

Syntax:

ip [ OPTIONS ] OBJECT { COMMAND | help }

Options:

  • address. 查询设备上的ip信息

    ip address
    ip address show {interface} // 查询指定设备的地址信息

  • link. 展示连接层信息

    ip link
    ip -s link // 展示网络接口的统计信息
    ip [-s] link show {interface}

  • route. 查询路由信息

    ip route

  • add. 为接口添加ip地址

    ip a add {ip_address} dev {interface}

  • del. 删除接口上的ip地址

    ip a del {ip_address} dev {interface}

  • up. 使能网络接口

    ip link set {interface} up

  • down. 关闭网络接口

    ip link set {interface} down

  • monitor. 在屏幕上持续展示实时的设备，地址和路由信息

    ip monitor

  • neighbor. 查看邻居的mac信息

    ip neighbor
    ip neighbor del {ip_address} dev {interface}
    ip neighbor add {ip_address} dev {interface}
